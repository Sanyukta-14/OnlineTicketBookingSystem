# Online Ticket Booking System

## Technologies
- React
- Spring Boot
- MySQL
- Docker
- Docker Compose
- Jenkins
- GitHub
- Vercel

## Features
- User Management
- Event Management
- Booking Management
- Ticket Cancellation
- Responsive UI

## CI/CD
- Jenkins builds the project automatically.
- Docker containers are built and started using Docker Compose.

## Deployment
Frontend: Vercel
Backend: Spring Boot
Database: MySQL
